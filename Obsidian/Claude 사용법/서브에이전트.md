# 서브에이전트 생성

1. Claude Code 에서 /agents 명령 입력
2. 서브 에이전트 관리 화면이 열리면 메뉴에서 Create new agent를 선택 
3. 서브 에이전트를 저장할 위치 선택
4. 서브 에이전트를 어떻게 만들지 선택
   * Generate with Cluade : 역할을 자연어로 입력하면 Claude가 설정 파일을 자동으로 작성
   * Manual configuration : YAML/마크다운 파일을 직접 작성하는 고급 방식
5. 서브 에이전트가 어떤 일을 할지 설정
6. 서브 에이전트가 사용할 수 있는 도구 설정
7. 모델 선택


# 서브에이전트 사용

1. 자동 실행
   AI가 작업을 맥락과 서브에이전트의 Description 필드를 분석해 적절한 서브에이전트를 자동으로 선택

2. 명시적 호출
   서브에이전트를 명시적으로 지정해 실행하는 방식으로, 프로덕션 워크플로에 권장됨

```
> code-reviewer 서브에이전트로 payment.js를 분석해줘
> test-writer를 사용해서 테스트 코드를 만들어줘.
```
