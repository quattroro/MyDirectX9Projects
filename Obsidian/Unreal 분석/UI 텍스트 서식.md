Format Text함수는 Format이라는 레이블을 가진 텍스트 입력 1개를 받고, Result 텍스트 출력을 반환한다. Format Text 함수는 UE_LOG처럼 %형식 지정자를 사용하는 대신, {}기호를 사용해 문자열에 전달할 수 있는 파라미터를 나타낸다 {} 기호 사이에는 필요한 파라미어터의 이름을 추가할 수 있다. 
Format Text함수는 Byte, Int, Float, Text 또는 EText Gender 변수 유형만 지원한다. 따라서 다른 변수 유형을 이 함수의 파라미터로 전달하려면 지원하는 유형 중 하나로 변환해야 한다.
