UE5에는 Game Only, Game And UI, UI Only라는 세 가지 입력 모드가 있다. 입력 모드가 Game을 포함하면, 플레이어 캐릭터와 플레이어 컨트롤러가 입력 액션을 통해 입력을 받는다. 입력 모드가 UI를 포함하면 화면에 배치된 위젯이 플레이어로부터 입력을 받는다. 화면에 이 위젯을 보여줄 때는 플레이어 캐릭터가 어떤 입력도 받지 않도록 해야 한다.
``` c++
SetInputMode(FInputModeUIOnly());
```
이렇게 한 다음 마우스 커서를 보여줘 플레이어가 마우스로 버튼을 선택할 수 있도록 한다.
``` c++
bShowMouseCursor = true;
```
